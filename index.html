Transition

  • What are transitions?

A transition is the smooth and gradual change of something from one state to another over time. It makes changes look natural instead of sudden.Conversion of an element from one state to another with smooth effects

  • Types of Transitions:

    1. Transition Property
    2. Transition Duration
    3. Transition Timing Function
    4. Transition Delay
    5. Combination Of Transition Properties

  1. Transition Property

    2. Defines which CSS property will go under transition

    .tpbox1{
    transition: background-color;
    }
    .tpbox1:hover{
    background-color: #bb3a7f;
    color: wheat;
    }
    .tpbox3{
    transition: font-size;
    }
    .tpbox3:hover{
    font-size: 2.3rem;
    }
    Background Changes
    Height &
    width Increases
    Text Transform
    Opacity

  2. Transition Duration

    3. This property defines that for how much time will our transition last

    .tdbox1{
    transition-property: background-color color;
    transition-duration: 0.5s;
    }
    .tdbox1:hover{
    background-color: #327abd;
    color: azure;

  3. Transition Timing Function

    4. Defines how the transition progresses over time

    .ttfbox2{
    transition: width ease-in 2s;
    }
    .ttfbox2:hover{
    width: 500px;

  4. Transition Delay

    5. Defines the wait time before the transition starts

    .tdebox2{
    transition-duration: 2s;
    transition-delay: 2s;
    transition: transform background-color ease-in-out ;
    }
    .tdebox2:hover{
    transform: rotateY(90deg);
    background-color: rgb(0, 152, 152);
    }

  5. Transform
